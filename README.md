# Hanna Bae Personal Homepage

Static GitHub Pages homepage for Hanna Bae.

## How to publish

1. Create a new GitHub repository named `hanna-bae.github.io` or use an existing GitHub Pages repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages** and set the branch to `main` and folder to `/root`.
4. Replace `assets/profile.svg` with a real profile photo if desired.
5. Update publication links when papers become public.

## Notes

- The News tab is intentionally omitted.
- `assets/CV_Hanna_Bae.pdf` is copied from the provided CV. Check the PDF before publishing because it contains personal contact information.
- The page is inspired by academic homepage layouts, but the HTML/CSS here is newly written for this homepage.
